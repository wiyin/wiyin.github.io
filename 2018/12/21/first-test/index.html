Oracle Linux Boot Process

Oracle Linux 7

  1. BIOS performs POST
  2. BOST read MBR for bootloaded
  3. GRUB 2 bootloader load vmlinuz kernel image
  4. GRUB 2 extracts the contents of the initramfs image
  5. The kernel loads the driver modules from initramfs
  6. Then kernel stat the system’s first process systemd
  7. The systemd process takes over, it
    • Read configuration files from the /etc/systemd folder
    • Reads the file linked by /etc/systemd/system/default.target
    • Brings the system to the state defined by the system targetk
    • Executes /etc/rc.local
